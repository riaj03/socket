# Socket Server

---

## Overview of Project Base Technologies Used

Javascript, Socket.io

## Setup Steps

1. Clone `socket-server` repository:
    - `https://github.com/grootsinc/socket-server.git`
2. Install dependencies:
    - run `yarn install`

```bash
    PORT=3009
    AUTH_TOKEN=
```

3. Start local environment server:
    - `yarn start`
